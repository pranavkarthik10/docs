---
title: "Get Event"
api: "GET /v1/events/{id}"
description: "Retrieve a single event by ID"
---

Retrieves the details of an existing event.

<ParamField path="id" type="string" required>
  The event ID (e.g., `evt_1a2b3c4d`)
</ParamField>

<RequestExample>
```bash cURL
curl https://api.flux.dev/v1/events/evt_1a2b3c4d \
  -H "Authorization: Bearer sk_live_..."
```

```typescript Node.js
const event = await flux.events.get('evt_1a2b3c4d');
```

```python Python
event = flux.events.get("evt_1a2b3c4d")
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": "evt_1a2b3c4d",
  "type": "order.completed",
  "created": "2024-01-15T10:30:00Z",
  "data": {
    "orderId": "ord_8x7kj2",
    "amount": 9900,
    "currency": "usd"
  },
  "deliveries": [
    {
      "webhookId": "wh_abc123",
      "status": "delivered",
      "statusCode": 200,
      "deliveredAt": "2024-01-15T10:30:01Z"
    }
  ]
}
```

```json 404
{
  "error": {
    "code": "not_found",
    "message": "Event not found"
  }
}
```
</ResponseExample>
